# EB Digital Studio V5.7 — Copilot OS

Bu sürümde EB Asistan, daha güçlü bir kontrollü satış zekâsına yükseltildi.

## Yeni özellikler
- Serbest müşteri mesajından sektör, şehir, hedef, modül ve tasarım dili algılama
- Fırsat skoru, risk skoru ve proje netlik skoru
- Executive Project Brief görünümü
- Olmazsa olmaz modül listesi
- Eksik bilgi analizi
- Sektörel SEO hedefleri
- Uygulama yol haritası
- Müşteri itirazlarına hazır cevaplar
- Premium ilk temas mesajı üretimi
- WhatsApp'a hazır proje özeti
- Teklif formunu otomatik doldurma

## Not
Bu sürüm API'siz kontrollü modda çalışır. Gerçek OpenAI bağlantısı için backend/proxy kurulumu gerekir; API anahtarı statik site içine koyulmamalıdır.
