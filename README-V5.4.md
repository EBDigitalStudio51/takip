# EB Digital Studio V5.4 — Distinct Style Engine

Bu sürüm V5.3 tabanı üzerine hazırlanmıştır.

## Değişiklikler
- Header ve footer EB logosu tıklanınca sayfanın en üstüne yumuşak biçimde döner.
- Footer `YUKARI ↑` bağlantısı güvenilir JavaScript kaydırmasıyla çalışır.
- Design Lab tasarım dilleri tamamen ayrıştırıldı:
  - Futuristik: neon grid, veri kartları, yörüngeler ve keskin butonlar
  - Lüks: editorial serif tipografi, altın detaylar, kemerli vitrin kompozisyonu
  - Minimal: beyaz zemin, yüksek boşluk, monokrom tipografi ve sade geometri
  - Kurumsal: açık kurumsal arayüz, güven veren mavi yapı ve dashboard grafiği
- Ön izleme başlığı seçili tasarım dilini gösterir.
- Cache sürümü 5.4 olarak güncellendi.
