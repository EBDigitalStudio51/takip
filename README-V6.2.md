# EB Digital Studio V6.2 — Cursor Fix

Bu güncelleme V6.1 Crown AI panelindeki imleç görünmeme sorununu düzeltir.

## Düzeltilenler
- Crown AI paneli site imlecinin üstüne çıkıp custom cursor’ı gizlemeyecek şekilde katmanlar düzenlendi.
- Asistan z-index değeri site üstünde kalacak ama cursor katmanını ezmeyecek seviyeye çekildi.
- Custom cursor, AI paneli üzerinde de görünür hale getirildi.
- Custom cursor desteklenmeyen cihazlarda native cursor fallback bırakıldı.
- Kapama butonu ve kompakt panel davranışı korundu.
