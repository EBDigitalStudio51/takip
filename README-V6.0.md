# EB Digital Studio V6.0 — EB Crown AI

Bu sürüm EB Asistanı **Client Winning Intelligence OS** seviyesine yükseltir.

## Yeni yetenekler

- Crown Project Intelligence raporu
- 6 uzmanlı strateji kurulu: CEO Strategist, UX Director, SEO Architect, Sales Closer, Risk Guardian, Ops Manager
- Mini canlı ön izleme reçetesi
- Fırsat, dönüşüm, otorite, risk, karmaşıklık ve ROI sinyali skorları
- 30/60/90 günlük growth plan
- Mini pitch deck
- İtiraz laboratuvarı
- Risk ve kalite kontrol
- Teklif formunu otomatik doldurma
- WhatsApp proje özeti

## Not

Bu sürüm API kullanmaz. Aylık maliyet ve API anahtarı yoktur. Gerçek ChatGPT/OpenAI bağlantılı sürüm istenirse backend üzerinden güvenli şekilde bağlanmalıdır; API anahtarı doğrudan site dosyalarına yazılmamalıdır.
