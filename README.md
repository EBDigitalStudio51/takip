# Niğde Web Sitesi Potansiyel İşletmeler Paneli

- `index.html` dosyasını tarayıcıda açın.
- Arama, kategori ve durum filtrelerini kullanın.
- Görüşme durumu ile notlar tarayıcının localStorage alanında tutulur.
- CSV indir butonu, güncel durum ve notları dışa aktarır.
- Veriler 17 Haziran 2026 tarihli kamuya açık web taramasına dayanır.
- İletişime geçmeden önce telefon ve web sitesi durumunu yeniden doğrulayın.
